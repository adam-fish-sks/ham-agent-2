# Device Classes

This document defines the classification of devices used in the HAM Agent system.

## Windows Devices

### Standard Windows
**Definition**: Entry-level business laptops with standard specifications
- **Processors**: Intel i5, i7, or Ultra 7 (without discrete graphics)
- **Memory**: Up to 16GB RAM
- **Graphics**: Integrated graphics only
- **Examples**: Dell Latitude 5520, Dell Latitude 5530, similar business-class laptops

### Enhanced Windows
**Definition**: High-performance workstations with advanced specifications
- **Processors**: High-end Intel i7, i9, or Ultra processors
- **Memory**: More than 16GB RAM (typically 32GB+)
- **Graphics**: Discrete graphics cards (NVIDIA, AMD)
- **Examples**: Dell Precision workstations, Dell XPS series, Dell Pro Max laptops

## Mac Devices

### Standard Mac
**Definition**: Entry-level and older generation MacBooks
- **Models**: MacBook Air (all generations)
- **Legacy Models**: MacBook Pro models from 2020 and earlier (pre-M1)
- **Processors**: Intel-based or base-level Apple Silicon (M1, M2, M3, M4, M5)
- **Memory**: Up to 16GB RAM

### Enhanced Mac
**Definition**: High-performance MacBook Pro models
- **Models**: MacBook Pro (2021 and newer)
- **Processors**: Apple Silicon M3, M3 Pro, M3 Max, M4, M4 Pro, M4 Max, M5, M5 Pro, M5 Max
- **Memory**: 32GB RAM or higher
- **Features**: Advanced GPU cores, ProMotion display, enhanced thermal performance

---

## Classification Guidelines

When classifying a device:
1. Identify the operating system (Windows or Mac)
2. Check processor and memory specifications
3. For Windows: Verify presence of discrete graphics
4. For Mac: Check generation and chip configuration
5. Assign appropriate class based on criteria above
